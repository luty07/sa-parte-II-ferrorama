O RailTrack é um aplicativo de gestão ferroviária desenvolvido para a Situação de Aprendizagem, com foco em cidades inteligentes. A ideia é criar um sistema para melhorar o transporte ferroviário usando tecnologia, deixando tudo mais eficiente, seguro e sustentável. O app permite que operadores acompanhem trens, rotas, manutenções e alertas em tempo real. Primeiro, fizemos os mockups das telas mobile, seguindo Mobile First e priorizando uma boa UX/UI. Neles, incluímos um dashboard geral, gestão de rotas, monitoramento de manutenção, relatórios e sistema de alertas. Depois, partimos para a implementação em HTML, CSS e JavaScript/PHP. A primeira tela criada foi a de login, com validação de e-mail e senha em JavaScript/PHP, exibindo mensagens de erro quando necessário. As outras telas seguiram o visual planejado nos protótipos. Também tivemos que fazer relatórios individuais, avaliando melhorias no design e no código, analisando organização visual, clareza, usabilidade e boas práticas. Outro relatório foi feito com base nas atividades práticas da aula, reforçando a importância de revisar e melhorar sempre o sistema. O RailTrack acabou sendo um exercício completo, juntando design, desenvolvimento web e análise crítica num contexto real de mobilidade urbana.
